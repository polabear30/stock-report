# 주식 에이전트 사용 가이드 (초보자용)

## 이 프로그램이 뭐하는 건가요?

우리가 만든 프로그램은 **AI가 미국 주식 시장을 매일 분석해 주는 로봇**이에요.
이 로봇이 하는 일은:

- 경제가 지금 좋은지 나쁜지 알아봐요 (경제 분석)
- 주식 차트를 보고 "지금 사야 할까? 팔아야 할까?" 판단해요 (기술 분석)
- 뉴스를 읽고 분위기가 좋은지 나쁜지 파악해요 (뉴스 분석)
- 이걸 종합해서 투자 전략을 알려줘요
- 위험하면 이메일로 알려줘요

---

## 1단계: 준비물 챙기기 (API 키 발급)

이 프로그램이 데이터를 가져오려면 **"열쇠(API 키)"**가 4개 필요해요.
마치 도서관 카드가 있어야 책을 빌릴 수 있는 것처럼요!

| 열쇠 이름 | 어디서 받나요? | 뭐하는 건가요? |
|-----------|---------------|---------------|
| ANTHROPIC_API_KEY | https://console.anthropic.com/ | AI 두뇌 (Claude) |
| POLYGON_API_KEY | https://polygon.io/ | 주식 가격 데이터 |
| FRED_API_KEY | https://fred.stlouisfed.org/docs/api/api_key.html | 경제 지표 데이터 |
| ALPHA_VANTAGE_API_KEY | https://www.alphavantage.co/support/#api-key | 뉴스 데이터 |

각 사이트에 들어가서 **회원가입** 하고 **API Key를 발급** 받으면 돼요.
(긴 영어+숫자 코드가 나와요. 예: `sk-ant-abc123...`)

---

## 2단계: 프로그램 설치하기

Cursor 터미널(아래쪽 검은 화면)을 열고, 아래 명령어를 **한 줄씩** 입력하고 Enter를 눌러요:

```bash
cd c:\cursor_test\stock
```

이건 "stock 폴더로 이동해!" 라는 뜻이에요.

```bash
pip install -r requirements.txt
```

이건 "프로그램이 필요한 부품들을 인터넷에서 다운받아!" 라는 뜻이에요.
(시간이 좀 걸릴 수 있어요. 기다려 주세요!)

---

## 3단계: 열쇠 입력하기 (.env 파일 만들기)

`stock` 폴더 안에 `.env.example` 이라는 파일이 있어요. 이걸 복사해서 `.env` 파일을 만들어야 해요.

터미널에 이렇게 입력해요:

```bash
copy .env.example .env
```

그 다음 `.env` 파일을 열어서, 1단계에서 받은 열쇠를 넣어요.
**바꿔야 할 부분만** 알려드릴게요:

```
ANTHROPIC_API_KEY=여기에_anthropic_열쇠_붙여넣기
POLYGON_API_KEY=여기에_polygon_열쇠_붙여넣기
FRED_API_KEY=여기에_fred_열쇠_붙여넣기
ALPHA_VANTAGE_API_KEY=여기에_alphavantage_열쇠_붙여넣기
```

`your_..._here` 라고 쓰여있는 부분을 지우고, 진짜 열쇠를 붙여넣으면 돼요!

---

## 4단계: 실행하기! (3가지 방법)

### 방법 A: "지금 바로 분석해줘!" (가장 추천)

```bash
cd c:\cursor_test\stock
py -3 main.py --run-now
```

AI가 알아서 경제 상황을 보고, 주식을 분석하고, 보고서를 만들어 줘요.
결과가 화면에 쭉~ 나와요.

### 방법 B: "매일 자동으로 분석해줘!" (스케줄러 모드)

```bash
cd c:\cursor_test\stock
py -3 main.py
```

이러면 프로그램이 **매일 미국 장 마감 시간(한국 새벽 5:30)**에 자동으로 분석을 해요.
프로그램을 끄고 싶으면 `Ctrl + C`를 누르면 돼요.

### 방법 C: "특정 종목만 분석해줘!"

```bash
cd c:\cursor_test\stock
py -3 main.py --run-now --prompt "NVDA와 TSLA를 집중 분석해 주세요"
```

따옴표 안에 원하는 내용을 적으면 돼요. 마치 AI에게 말하는 것처럼!

---

## 5단계: 대시보드 보기 (예쁜 화면)

분석 결과를 예쁜 차트와 표로 보고 싶으면:

```bash
cd c:\cursor_test\stock
streamlit run dashboard/app.py
```

그러면 웹 브라우저가 자동으로 열리면서 이런 것들을 볼 수 있어요:

- 내 포트폴리오 현황 (얼마나 벌었는지/잃었는지)
- 일별 자산 변화 차트
- AI가 작성한 일일 보고서

## 자주 묻는 질문

**Q: 관심 종목을 바꾸고 싶어요!**
`.env` 파일을 열고 `WATCHLIST=` 뒤에 원하는 종목을 쉼표로 나열하면 돼요.
예: `WATCHLIST=AAPL,NVDA,TSLA,META`

**Q: 이메일 알림을 받고 싶어요!**
`.env` 파일에서 이메일 설정을 해야 해요:
- `SMTP_USER=내이메일@gmail.com`
- `SMTP_PASSWORD=구글앱비밀번호` (Gmail 설정에서 "앱 비밀번호" 발급 필요)
- `ALERT_RECIPIENTS=알림받을이메일@gmail.com`

**Q: 가상 투자금을 바꾸고 싶어요!**
`.env` 파일에서 `INITIAL_CASH=100000.0` 숫자를 바꾸면 돼요.
(기본값은 10만 달러예요)

**Q: 에러가 나요!**
대부분 API 키가 잘못 입력되었을 때 에러가 나요.
`.env` 파일을 다시 확인해 보세요. 열쇠 앞뒤에 빈칸이 있으면 안 돼요!
